# PolicyLens Demo Examples

This directory contains lightweight fictional files for the v0.1.0-alpha demo workflow.

- `demo-research-article.txt`: a short Chinese energy and market research article.
- `demo-policy-notice.txt`: a fictional policy notice for local demonstration only.
- `demo-policy-metadata.json`: metadata to use when ingesting the demo policy.
- `demo-workflow.http`: HTTP request sequence for VS Code REST Client or similar tools.

The demo policy is not an official policy document and must not be cited as real policy evidence.
